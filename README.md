<h1>Projet Final – Carnet de téléphones (Java)</h1>

<h2>Description</h2>
<p>
Petit projet <strong>Java</strong> permettant de gérer un carnet de téléphones
avec validation des données et tests unitaires.
</p>

<h2>Fichiers</h2>
<ul>
  <li>
    <strong>CarnetTelephones.java</strong> :
    logique principale du carnet
  </li>
  <li>
    <strong>Telephone.java</strong> :
    modèle / objet téléphone
  </li>
  <li>
    <strong>TelephoneInvalideException.java</strong> :
    exception personnalisée
  </li>
  <li>
    <strong>TestsCarnetTelephones.java</strong> :
    tests
  </li>
</ul>
